# 技术栈说明

## 后端框架

- **Spring Boot 3.2.5**
    - 快速构建 RESTful 服务
    - 内置自动配置，降低基础设施代码

## 持久化与数据库

- **Spring Data JPA**
    - 使用仓储接口简化 CRUD
    - 降低 SQL 样板代码
- **MySQL 8+**
    - 结构化数据存储
    - 适合学分、活动、统计等关系型场景

## 校验与开发效率

- **Spring Validation**
    - 统一请求参数校验
    - 在入口层保障数据合法性
- **Lombok**
    - 简化实体与 DTO 模板代码
    - 提升开发效率

## 构建与运行

- **Maven**
    - 依赖管理与构建打包
- **Java 17**
    - 长期支持版本，兼容 Spring Boot 3.x

## 当前架构原则实践

- **KISS**：仅实现学分统计核心主链路
- **YAGNI**：暂不引入消息队列、缓存、分布式事务
- **DRY**：统一响应结构与异常处理，避免重复控制器代码
- **SOLID**：控制器、服务、仓储职责清晰分层
